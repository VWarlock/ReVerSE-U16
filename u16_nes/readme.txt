Nintendo Entertainment System (NES) for ReVerSE-U16 Rev.C board (https://github.com/mvvproject/ReVerSE-U16)
Ported from https://github.com/mist-devel/mist-board/tree/master/cores/nes

Please use:
	- release/u16_nes.jic - for FPGA programming
	- vnc2/build/release/firmware.rom - for vnc2 programming

USB HID joysticks or Win + Up, Down and LShift for select new games.

Special/console keys:
	ESC  	- Reset
	Win 	- OSD Menu
	Enter 	- Start
	Space 	- Select
	Cursor 	- D-Pad (Up, Down, Left, Right)
	LShift 	- D-Pad (B)
	LCtrl 	- D-Pad (A)

Joysticks support:
	Definder Game Master G2
